Fun Fun Function


The fun fun function videos are pretty great. Here are some of the things I've learned:

  • .filter
    • .filter() will take an array and return only the items that fit the given criteria.
    • It does this by assigning true or false values to the items in the aray in order to decide whether to include them in the new array.
  • .map
    • .map() will take an array and return all the items in the array, modified by the given criteria.
    • It does this by taking only part of the item, or modifying it in whatever way that you specified.
  • .reduce
    • .reduce() is a multi-purpose version of the above higher order functions.
    • It can be used when either .map or .filter don't quite fit the problem
    • It needs a bit more info to work, but can replace many for loops.

This

"this" seems to have many purposes, kind of dependent on where in your code you use it.
It behaves slightly differently dependig on whether you have "use strict" set, and whether or not you are using arrow functions.
"this" generally referrs to the scope of whatever it is being called in, which means that the value it returns can change if you try to call anything using "this" while outside of its native object.
You can get around that issue by using "bind" to set the object of "this" back to its original object.
